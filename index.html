Viewers Choice Presentation

This page discusses about Life Expectency and Probability of Death visualization developed by Institute for Health Metrics and Evaluation. You can check the viz on this link.

Description

This visualisation is used for learning more about life expectency, how it has changed over the years 1990 - 2013. Also you can learn about various causes of death, what are the most probable way people die in specific region/country. You can also compare among countries and find out information like how different are causes of death in developed countries compared to developing or under-developed countries.

Data

Data is about what are the different causes of death. It contains information collected from 1990 to 2013 and has information about all the countries. You can download the datasets from this link. Each file is around 30MB, stored in CSV format. You can view sample data in the screenshot. Most of the data is collected from surveys run in different parts of the world. Collecting information at this particular level is really difficult since the data is not only about cause of death in different coutries but also it is for a long period i.e. from 1990 to 2013. Also surveys were the first source for information since many of those countries doesn't have these kind of data readily available. Also since the data is about all the countries data collected might be of various formats and integrating them together is another challenging job.

Features

Life Expectency

View how life expectency changed from 1990 to 2014 on either map or chart.

Cause of Death

View different causes of death and how it changed from 1990 to 2014 in the form of map and chart

Play/Pause

Play/Pause functonality to view how countries improved in case of life expectency.

Filter / Sort / Group Data

Filter, sort data based on gender, cause of death, year and location. Also group data by ages.

Detailed Information

Get deatailed information about different causes of death in a single click.

Good Things about the Viz

Color Palette

Overall color palatte is good. Colors don't look that bright and each color has some meaning. For example: When there is some improvement in the life expectency, the color was moving towards blue and when the life expectency was decreasing, color was moving towards red.

Controls

There are controls for locking down the scale which is useful when we are comparing data over different years. Also we have Play/Pause button which helps us and makes it easier for us to identify which countries/regions are moving towards blue color and which are moving towards red.

Charts

There are charts which helps us pick few countries of our interest and compare data among them. This makes sense when we want to do direct comparision instead of looking at the maps and keep track of those colors.

Information at Single Location

When we click on each bar in life expectency chart, detailed description on all the different causes of death is given at one single place which is good representation since we don't want to move around the screen too much to get information about different causes of death.

That could have been Better

Poor Design on Charts Page

Using of legend to filter the content is very bad design. Unless they are moving mouse over entire screen it is very difficult for people to know that there is a legend over there and it has some funcitonality built in it.

Inconsistent Functionality

There is hover functonality in one page and click on another. In general it is better to use click since that would be helpful on touch displays as well.

Dropdown Position

Drop down which contains variosu types of causes of death is not placed properly - Many might miss it because it was not there in life expectency.

Not Giving User a Choice

There are seperate charts for male and female in causes of death, but it would have been better if we had both option as well and let the user decide which charts he want to see.
